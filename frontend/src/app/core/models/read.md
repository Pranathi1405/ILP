```md
# Models

## Purpose

Models define TypeScript interfaces for application data.

They provide:

- Strong typing
- IntelliSense
- Safer refactoring
- Better code readability

## Why Important?

Models:

- Prevent runtime errors
- Make backend contracts clear
- Improve scalability
- Document data structures

## Examples

- Test model
- Question model
- User model
- Result model

## Best Practices

- Keep models clean (no logic inside)
- Match backend API contracts
- Use union types where needed
- Avoid `any`
```
